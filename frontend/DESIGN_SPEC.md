# Lull · 听简 — 设计规范 (Design System)

70 年代复古迷幻海报风 × 英语分级阅读。导航 / 头部 / 空状态狂野撞色，正文阅读区克制易读。

---

## 1. 品牌 Brand

| 项目 | 值 |
|------|----|
| 产品名 | **Lull · 听简** |
| Logo | `Lull`（Shrikhand 斜体 + 黑色硬阴影）· 圆点分隔 · `听简`（宋体衬线） |
| 主图形 | 六瓣花朵（`Flower.dc.html`，可调花瓣色 / 花心色 / 尺寸） |
| 气质 | 大胆、明快、手作感；黑描边 + 硬投影的「新粗野 × 复古」混血 |

---

## 2. 色彩 Colors

所有彩色块统一 **2.5px 黑色描边** (`#1a1817`) + **硬投影**（无模糊，向右下偏移 3–9px）。

| 角色 | HEX | 用途 |
|------|-----|------|
| 米底 Cream | `#f2ecda` | 页面背景 |
| 卡片 Panel | `#fffdf6` | 卡片 / 输入框底 |
| 墨黑 Ink | `#1a1817` | 描边、正文强调、文字 |
| 桃粉 Pink | `#e34e98` | 主强调之一、徽章、雷达 |
| 橘橙 Orange | `#ec6a2b` | 主 CTA、生词高亮、Logo |
| 蓝 Blue | `#2f7fd6` | 次强调、A 级难度、整句翻译开 |
| 黄 Yellow | `#f3c44b` | 高亮、测验区底、激励态 |
| 绿 Green | `#1f9d5c` | 正确 / 已掌握 / 成功态 |
| 红 Red | `#d94646` | 错误 / 退出 / 危险态 |
| 正文 Body | `#26241f` | 阅读区文字 |
| 辅助文字 | `#5c5749` / `#7a7568` | 说明、次要信息 |
| 浅描边 | `#d8cfb4` / `#cbc09f` | 虚线分隔、雷达网格 |

> 规则：撞色只用于「外壳」（导航、头部、卡片头、徽章、空状态）。阅读正文一律墨黑 on 卡片底，保证长时间可读。

---

## 3. 字体 Typography

通过 Google Fonts 载入；中文走系统衬线/黑体回退。

| 角色 | 字体 | 用法 |
|------|------|------|
| 展示标题 Display | **Shrikhand**（斜体） | 大标题、品牌、数字大字。常配 `text-shadow:3px 3px 0 #1a1817` |
| 界面 / 标签 UI | **Space Grotesk** 400–700 | 按钮、导航、徽章、表单、图例 |
| 阅读正文 Reading | **Newsreader**（衬线，含斜体） | 文章正文、例句、词义；`line-height:1.8–1.9`，正文 19–20px |
| 中文点缀 | 系统宋体（`Songti SC / Noto Serif SC`） | 「听简」等小字中文 |
| 等宽 Mono | `ui-monospace`（系统） | 音标可选 |

**尺寸基线**：大标题 `clamp(38px,6vw,64px)`；卡片标题 21–26px；正文 19.5px；说明 13–14px；徽章 11–13px。移动端正文不小于 16px。

## 4. 形状与材质 Shape & Material

| Token | 值 |
|-------|----|
| 描边 | `2.5px solid #1a1817`（细节 2px） |
| 硬投影 | `Npx Npx 0 #1a1817`，N = 3（按钮）/ 5（卡片）/ 6–9（大卡片、登录） |
| 圆角 | 徽章/按钮 `999px`；输入 12–14px；卡片 16–22px；难度标签 999px |
| 棋盘格 | `conic-gradient(#0001 0 25%,transparent 0 50%,#0001 0 75%,transparent 0)`，`background-size:34–60px` |
| 悬停 | 卡片 `transform:translate(-2px,-2px)` + 投影加深到 8px |

---

## 5. 组件 Components

- **主按钮**：`background:#ec6a2b;color:#fff;border:2.5px;box-shadow:4px;border-radius:14px`
- **次按钮 / Chip**：白底墨边药丸；选中态填充对应强调色
- **难度徽章**：黑底白字药丸（A1–C2）
- **掌握度标签**：学习中=橙 / 复习=蓝 / 已掌握=绿
- **卡片**：白底 + 墨边 + 硬投影；常带彩色「卡头」（图标大字 + 棋盘格）
- **开关 Toggle**：药丸轨道（开=绿）+ 白色圆钮滑动
- **雷达图**：5 维五边形（阅读/词汇/语法/坚持/准确），网格 4 环 + 数据多边形（粉描边半透明填充）+ 彩色顶点 + 图例进度条
- **空状态**：虚线框 + 旋转花朵 + 一句引导文案

## 6. 可调参数 Tweaks（根组件 props）

| prop | 类型 | 默认 | 作用 |
|------|------|------|------|
| `showFlowers` | boolean | `true` | 背景旋转花朵开关 |
| `primaryAccent` | enum(orange/pink/blue) | `orange` | 主色调 |

`Flower.dc.html` props：`petal`（花瓣色）、`center`（花心色）、`size`（px）。

---

## 7. 版式节奏 Layout

- 内容最大宽 `1180px`，左右留白 24px。
- 阅读区正文宽度受卡片约束（约 680px），保证每行字数适中。
- 卡片网格 `repeat(auto-fill,minmax(290px,1fr))`，gap 18–22px。
- 一切元素用 flex/grid + `gap`，不用裸边距堆叠。
